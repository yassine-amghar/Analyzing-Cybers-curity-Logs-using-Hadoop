# Analyzing-Cybers-curity-Logs-using-Hadoop
Big Data cybersecurity project analyzing logs (50M events) using Hadoop, Spark &amp; Kafka. Detects brute force attacks (10M failed logins, 500 IPs), port scans, and SQL injections (650K attempts) in &lt;30 mins on Docker cluster. Provides visualizations and security recommendations for enterprise threat detection.
